## Project Overview

This project is the final and most complete version of my personal portfolio web application developed throughout SWE 363.

The website was gradually expanded across previous assignments, beginning as a structured portfolio website, then evolving into an interactive front-end application, and finally becoming a polished, professional, API-driven personal web application.

Assignment 4 focuses on bringing all previous work together into one complete and production-ready system that demonstrates:

- strong technical implementation
- clean and responsive design
- advanced JavaScript interaction
- external API integration
- documentation quality
- AI-supported development
- innovation and creativity
- professional presentation readiness

This final version is designed to feel like a realistic modern personal portfolio rather than a basic classroom webpage.

## Main Objectives of the Project

The goals of this project are:

- present personal and academic information in a professional way
- demonstrate front-end web development skills
- show practical JavaScript interactivity
- connect the interface with live external data
- provide a user-friendly browsing experience
- include creative features that improve engagement
- showcase the complete learning journey from earlier assignments

---

## Main Features Included

### 1. Responsive Professional Landing Page
- dynamic greeting based on current time
- animated counters
- personalized visitor interaction
- smooth call-to-action buttons

### 2. Personalized Visitor Experience
Visitors can:
- enter their name
- save it using localStorage
- receive a personalized greeting
- clear stored personalization

### 3. Interactive Projects Showcase
The Projects section includes:
- project category filtering
- live keyword search
- dynamic results count
- favorite project saving using localStorage
- modal popup for detailed project information

### 4. GitHub API Integration
The website fetches live GitHub data including:
- GitHub profile avatar
- profile name and username
- biography
- followers / following / repositories
- latest repositories
- repository language, stars, forks, update date

### 5. Quote API Integration
Displays:
- random live motivational quote
- author name
- refresh quote button

### 6. Weather API Integration
Displays current live weather in Al-Khobar:
- temperature
- humidity
- wind speed
- weather condition
- last updated timestamp

### 7. Professional Journey Timeline
A dedicated visual timeline explains how the portfolio evolved from:
- basic structure
- interactive UI
- API integration
- final polished application

### 8. Resume Access Section
Users can:
- view resume
- download resume PDF

### 9. AI Portfolio Assistant
A custom interactive assistant provides guided responses about:
- the student
- included projects
- technologies used
- responsible AI usage
- future improvements

### 10. Skills Tab System
Visitors can switch between:
- programming skills
- technical tools
- soft skills

### 11. Contact Form with Validation
The form includes:
- required field checking
- email validation
- message length validation
- error highlighting
- professional submission simulation

### 12. Additional UX Enhancements
- dark/light theme toggle
- scroll progress bar
- scroll-to-top button
- reveal animations
- active navigation highlighting
- mobile responsive menu
